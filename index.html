About the App

Linkup is a real-time Android chat application that allows users to exchange messages with real-time sent/seen status. Built using Java, XML, and Firebase, Linkup delivers a seamless chatting experience.

  • Login/Authentication
  • Real-time messaging with sent/seen indicators
  • Calling feature (coming soon)
  • Modern UI with RecyclerView
